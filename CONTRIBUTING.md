# Contributing

Thanks for your interest! Here's how to get involved:

## Small changes and new links — open a Pull Request

Good candidates for a direct PR:

- Adding a project, paper, talk, or blog post to an existing section
- Fixing a typo or broken link
- Improving a one-line description

Keep entries concise: a name, link, and one-line description is enough. Place new entries in the most relevant existing section.

## Suggesting new sections or structural changes — open an Issue

If you'd like to propose a new section, a significant reorganization, or a new supporting document, open an issue first to discuss it. This avoids wasted effort and helps align on scope before a PR.

## Questions for the FAQ — comment on Issue #5

Head to [Issue #5](https://github.com/raymyers/awesome-verified-coding-agents/issues/5) and comment with any question about the combination of AI coding agents and Formal Methods. Answering your own question (or others') is encouraged.
